oodba domain

Create temporary PED EA/DA admin accounts (breakglass account)

sl C:\PED\SCRIPTS\

.\BulkADUserCreationToSpecificOU.ps1 -csv C:\PED\CSVs\oodba_create_temp_users.csv

Copy-Item C:\PED\SCRIPTS\Fresh_new_accounts.csv C:\ped\CSVs\oodba_temp_users.csv

Grant EA/DA admin rights to temporary admin accounts

sl C:\PED\SCRIPTS\

.\AddObjectsToMultipleGroups.ps1 -CSV C:\PED\CSVs\oodba_temp_users_perm.csv

Deprivilege all other EA/DA/Tier0 admin user accounts (to be replaced by new Tier 0 accounts later)

import-csv C:\PED\CSVs\oodba_old_admins.csv | %{set-aduser $_.samaccountname -Enabled:$false -SmartcardLogonRequired:$true}

DISABLE all Tier 0 service accounts (coordinated with service account owners)

import-csv C:\PED\CSVs\oodba_service_accs.csv | %{set-aduser $_.name -Enabled:$false}

Restart All DCs according to Reboot plan (start with PDC)

sl C:\PED\SCRIPTS\

.\RemoteServerRestart.ps1 -CSV C:\PED\CSVs\oodba_reboot_plan.csv

Verify Forest/Domain Trusts reset

sl C:\PED\SCRIPTS\

.\VerifyTrusts.ps1

Create and issue new Tier 0 admin accounts & passwords to administrators

sl C:\PED\SCRIPTS\

.\BulkADUserCreationToSpecificOU.ps1 -csv C:\PED\CSVs\oodba_create_new_admins.csv

Copy-Item C:\PED\SCRIPTS\Fresh_new_accounts.csv C:\ped\CSVs\oodba_new_admins.csv

Configure Tier 0 admins as "account sensitive and can't be delegated"

import-csv C:\ped\CSVs\oodba_new_admins.csv | %{set-aduser $_.name -AccountNotDelegated:$true}

Grant EA/DA admin rights to permanent admin accounts (and add them to "Protected Users")

sl C:\PED\SCRIPTS\

.\AddObjectsToMultipleGroups.ps1 -CSV C:\PED\CSVs\oodba_new_admins_perm.csv

Reset password on all Tier 0 service accounts

sl C:\PED\SCRIPTS\

.\ResetPasswordsFromFile.ps1 -csv c:\ped\CSVs\oodba_service_accs.csv -showpass

Copy-Item C:\PED\SCRIPTS\New_Passwords.csv C:\ped\CSVs\oodba_svc_acc_pass.csv

Enable Tier 0 service accounts

import-csv C:\PED\CSVs\oodba_service_accs.csv | %{set-aduser $_.name -Enabled:$true}

Remove all temporary accounts and permissions

sl C:\PED\SCRIPTS\

.\DeleteUsers.ps1 -csv C:\PED\CSVs\oodba_temp_users.csv

Verify Forest/Domain Trusts reset

sl C:\PED\VerifyTrusts\

.\VerifyTrusts.ps1
